## About

CASE FILE ARCHIVES is an independent paranormal research project dedicated to documenting, analyzing, and preserving some of the most discussed paranormal cases in modern history.

The objective is not to prove or disprove paranormal phenomena, but to examine available evidence, witness testimony, historical records, skeptical explanations, and paranormal interpretations in a structured and balanced manner.
